# 渔阳城设定

## 一、城市概览

### 基本信息
- **城市名称**：渔阳城
- **地理位置**：东域沧澜海岸北端，临海而建，是沧澜剑宗辖下最北端的边远小城
- **城市定位**：海防前哨、海产交易中心、冒险者集散地
- **人口规模**：约五万余人，其中修仙者约千人，凡人居民以渔农、矿工、商贩为主
- **城市特色**：因常年抵御海兽，民风彪悍，尚武之风浓厚；冒险者众多，鱼龙混杂；**丹道凋敝、一丹难求**——城中无像样丹师，低阶疗伤丹都靠外地商队流入、量少价高，修士受伤常只能硬扛，这是主角以丹道立足的最大红利窗口
- **通行法则**：修仙界本就拳头大者说了算、私斗各凭实力；城主府只理城池日常管理与资源收集，不主动插手势力纷争，唯有触及城池根本（城防安危、资源命脉、官方权威）才会出手。渔阳城临海尚武、冒险者鱼龙混杂，这股风气更盛

### 城市布局
```
          北（海岸线）
    ┌───────────────────┐
    │  渔港码头区      │
    │  海防城墙        │
    ├───────────────────┤
    │  东区·散修灵田区   │
    │  南区·陈家       │
    │  西区·庆家       │
    │  北区·城主府     │
    │  中心·坊市大街   │
    └───────────────────┘
          南（内陆）
```

- **北区**：城主府驻地，守备军大营，掌管城防和政务
- **东区**：城东灵脉所在，散修聚居与灵田区，灵气优于他处
- **南区**：陈氏家族盘踞，掌控城南码头和海产贸易
- **西区**：庆氏家族盘踞，掌控城西矿脉和山林资源
- **中心区**：坊市、酒楼、客栈、冒险者聚集区
- **北区外**：渔港码头，商船渔船停泊，海兽防线最前线

## 二、战力体系

### 战力天花板（金丹初期）
- **城主·周沧海**：金丹初期，年约五旬，出身沧澜剑宗外门，因伤退役后被派来镇守渔阳。性格沉稳，深谙制衡之道，以守城安民为己任

### 第二梯队（筑基期）
- **筑基圆满（2人）**：
  - **城主府副城主·韩烈**：筑基圆满，周沧海的副手，实战经验丰富，常年坐镇海防
  - **陈家老祖·陈天霸**：筑基圆满，陈家太上老祖，常年闭关，极少露面

- **筑基中后期（3-4人）**：
  - **庆家族长·庆远山**：筑基后期，庆氏现任家主，精明干练
  - **陈家族长·陈洪涛**：筑基中期，陈氏现任家主，性格刚烈
  - **城防军统领·赵刚**：筑基中期，常年驻守城墙

- **筑基初期（若干）**：各家族长老、客卿等，约十余人

### 第三梯队（练气期）
- **练气圆满~九层**：各势力中层骨干，约数十人
- **练气五六七八层**：普通修士、护卫、弟子，约数百人
- **练气一二三四层**：底层修士、杂役、散修，约数百人

## 三、主要势力

### 1. 城主府
**性质**：官方行政机构，直属沧澜剑宗辖下

**核心职责**：
- 渔阳城日常政务、税收、户籍等城池管理
- 资源收集与调配（海产、矿税、战略物资统筹），保障城池运转
- 组织城防，抵御海兽侵袭（核心生存职能）
- 对各方势力以制衡为主，于陈庆两族间刻意维持均势、坐收渔利；私斗与势力纷争一概不主动介入，唯有触及城池根本（安危、资源命脉、官方权威）才会出手
- 向上级（沧澜剑宗）汇报情况

**组织架构**：
- **城主**：周沧海（金丹初期），最高决策者
- **副城主**：韩烈（筑基圆满），分管城防
- **文武官署**：
  - 城防营：驻守城墙，抵御海兽，约五百人
  - 巡逻队：城内治安，约百人
  - 户曹：户籍、税收
  - 兵曹：军械、战备
  - 法曹：司法、刑狱

**实力特点**：
- 人数最多，但整体战力偏弱（以凡人军队为主）
- 装备精良，有制式法器和守城弩
- 背靠沧澜剑宗，关键时刻可求援
- 对各家族有节制权，但实际控制力有限

**与主角关系**：
- 初期无直接交集
- 后续通过**三方比武**（城主府为参赛三方之一，下场争海岸线资源份额）、城防任务、海兽防御战产生联系
- 周沧海深谙制衡，比武中多刻意不让城主府独占头名，借此把前两成大头留给陈庆争抢、维持城中均势

---

### 2. 本地修仙格局（无本土宗门）
**性质**：渔阳城是沧澜剑宗辖下的边远小城，**本地无修仙宗门**。城中修仙者分三类：

- **城主府修士**：周沧海（金丹初期）、韩烈（筑基圆满）及城防军中的修士，代表官方力量
- **家族修士**：陈、庆两大家族豢养的客卿与子弟
- **散修**：城中数量最多的底层修士，鱼龙混杂，多依附佣兵团或接散活为生

**格局特点**：
- 没有本土宗门坐镇，高阶战力稀薄，金丹期唯城主一人
- 散修抱团或依附势力，佣兵团是散修主要的组织形态
- 修炼资源靠进山出海自取，或向城主府、家族兑换，无宗门供给
- 城东灵脉一带为散修与家族共用的灵田区，灵气优于他处

---

### 3. 陈氏家族
**性质**：渔阳城两大家族之一，海商世家

**家族背景**：
- 祖籍渔阳城，世代以海产贸易和渔业为生
- 传了五代人，积累了丰厚家底
- 在城南码头势力最大，掌控近半海产交易

**家族架构**：
- **老祖**：陈天霸（筑基圆满），太上，常年闭关
- **家主**：陈洪涛（筑基中期），现任家主，性格刚烈
- **嫡系子弟**：
  - 大公子陈昊（练气八层）：沉稳，内定继承人
  - 二公子陈玉（练气七层）：纨绔，好惹事
  - 三小姐陈雨柔（练气五层）：聪慧，掌管家族部分生意
- **旁系长老**：练气后期~筑基初期若干
- **护卫客卿**：练气期数十人

**家族特色**：
- 优势：有钱，海上资源丰富，人脉广
- 劣势：高阶战力少，修炼资源靠外部供给
- 产业：码头、渔场、海产加工、商船队、坊市店铺

**与主角关系**：
- **陈家是前期主要对立面之一**：陈玉对陆渊的嫉恨是个人恩怨（拍卖会丢脸、攀上东江城丹阁），但真正忌惮陆渊、又怕他倒向庆家打破均势的是陈家上层（家主陈洪涛、老祖陈天霸）。两族竞争走"三方比武定资源"的明规则，不演成家族战
- 冲突主轴：陈玉寻衅 → 铁鹰团直接出面施压（私斗各凭实力，城主府不介入）或逼陆渊退出庆家比武战席 → 本届三方比武上陈家与庆家（陆渊为庆家第三战力）正面对撞，可出人命但不灭门
- 家族生意、海岛探索、商船护卫等均可作任务来源或冲突引子

---

### 4. 庆氏家族
**性质**：渔阳城两大家族之一，矿主世家

**家族背景**：
- 祖上是矿工出身，偶然发现灵矿后发家
- 掌控城西万兽山边缘的几处矿脉
- 在山林资源开采方面势力最大

**家族架构**：
- **家主**：庆远山（筑基后期），现任家主，精明干练
- **嫡系子弟**：
  - 大公子庆文（练气七层）：书生型，管理矿账
  - 二公子庆武（练气六层）：鲁莽，负责矿场安保
  - 大小姐庆兰（练气五层）：温婉，待字闺中
- **旁系长老**：练气后期若干
- **矿工护卫**：练气期数十人，凡人矿工数百

**家族特色**：
- 优势：矿脉资源稳定，收入丰厚，有实业
- 劣势：高阶战力不足，家族子弟整体素质一般
- 产业：铁矿、灵石矿（低品）、灵木采伐、药山、坊市店铺

**与主角关系**：
- 矿脉探索、山林任务的来源
- 矿难、妖兽袭矿等事件可作为剧情
- **与陈家的资源角力是前期主线冲突之一，走"三方比武"明规则**：庆家在陈家压迫下急需外援，本届三方比武缺一名筑基战力，遂将第三战席许给陆渊（丹道价值+筑基中期），使其成为庆家倚重的战力；陆渊亦可借庆家在城中的根基与陈家周旋
- 庆兰（练气五层）可作善缘支点，庆家是陆渊前期少数能倚靠的本土势力

---

### 5. 佣兵团/冒险者势力

**整体情况**：
- 渔阳城因海防、矿场、海贸需求，佣兵团和冒险者众多
- 大小佣兵团十几个，散修冒险者数百人
- 鱼龙混杂，有规矩也有黑吃黑

**主要佣兵团**：

#### 铁鹰佣兵团
- **团长**：铁鹰（练气九层）
- **人数**：三十余人
- **骨干**：练气五六层七八人
- **业务**：护卫、押运、猎杀妖兽
- **特点**：实力最强，行事霸道，是陈家明面上的打手与刀
- **与主角关系**：前期主要敌对势力之一；陈家不方便亲自下场时，多由铁鹰团直接出面或野外围杀出手

#### 黑风盗
- **首领**：未知
- **人数**：不详
- **业务**：打劫、暗杀、黑市交易
- **特点**：神出鬼没，无恶不作，官府通缉
- **与主角关系**：反派势力，可作为野外冲突来源

#### 野草佣兵团
- **团长**：大壮（练气八九层）
- **人数**：十余人（含主角后）
- **骨干**：阿福（练气五六层）、林轻语（练气三四层）
- **业务**：采集、低级护卫、小型猎杀
- **特点**：底层小团，日子艰难，但人心齐
- **与主角关系**：主角初期所在团队

#### 其他小佣兵团
- 猛虎团、血刀会、海狼队等，规模不一，各有擅长

## 四、周边资源

### 1. 海洋资源
**近海区域（0-50里）**：
- **渔获**：各类海鱼、海虾、海蟹，凡人渔民主要作业区
- **低阶海兽**：海蛇、毒水母、章鱼怪等，练气期一二层
- **海底资源**：普通珍珠、珊瑚、贝壳
- **岛礁**：无名小岛若干，多为海鸟栖息

**中海区域（50-200里）**：
- **灵贝养殖场**：几处大型灵贝礁，产灵珠
- **海草带**：各类海水灵草，是炼丹材料
- **中阶海兽**：海狮、海豹人、剑鱼群等，练气期三四层
- **珊瑚礁群**：盛产各类水系材料
- **沉船遗迹**：古代商船沉没处，偶有收获

**远海区域（200里以外）**：
- **深海矿脉**：传说有海底灵石矿，品质极高，但开采困难
- **高阶海兽**：海蛟龙、巨型章鱼等，筑基期以上
- **神秘海岛**：传闻有上古遗迹或秘境，但危险重重
- **迷雾海域**：常年被海雾笼罩，容易迷失方向

**海岛分布**：
- **渔火岛**：最近的岛屿，距城50里，有渔民村，是商船中转站
- **黑石岛**：距城百里，岛上有黑石矿脉，庆家势力范围
- **青螺岛**：距城百五十里，形似青螺，传有修士遗迹
- **无名荒岛**：众多，大小不一，部分有妖兽盘踞

---

### 2. 山林资源
**万兽山（城西方向）**：
- **外围（0-30里）**：
  - 常见野兽、低阶妖兽
  - 普通草药、灵草
  - 凡人猎人和采药人活动区
  
- **中部（30-100里）**：
  - 中阶妖兽：练气期三四层到七八层
  - 珍稀药材、灵木
  - 矿脉分布区（庆家掌控部分）
  - 佣兵团主要活动区
  
- **深处（100里以外）**：
  - 高阶妖兽：筑基期以上
  - 天材地宝传说
  - 极少有人敢深入
  - 据说有上古遗迹

**山林特产**：
- **药材**：凝气草、聚灵花、疗伤药草等（外围常见）
- **矿石**：铁精矿、铜精矿、低品灵石矿（中部分布）
- **妖兽材料**：兽皮、兽骨、兽丹（按等级价值不同）
- **灵木**：铁脊木、青钢木等（用于炼器和建筑）

---

### 3. 其他资源
**灵田**：
- 城东灵田三十亩（散修与家族共用），产灵谷
- 各家族有少量灵田
- 产量有限，仅供内部使用

**药园**：
- 城东药园，种植普通灵草
- 陈家和庆家各有小型药园
- 珍稀药材仍需进山或出海采集

---

## 五、秘境与遗迹

### 1. 青螺岛遗迹
**位置**：青螺岛地下
**传闻**：
- 据说是上古某位散修的洞府
- 入口隐蔽，需特定条件才能开启
- 每隔一段时间会有灵气波动，吸引冒险者前往
- 至今无人能找到真正的入口

**可能的剧情**：
- 主角偶然发现入口线索
- 陈家/庆家组织探索队
- 佣兵团接受护卫任务
- 遗迹内有机缘也有危险

### 2. 万兽山深处秘境
**位置**：万兽山深处某处
**传闻**：
- 传闻有古代宗门遗址
- 有强大妖兽守护
- 偶有异象传出
- 具体位置不明

**可能的剧情**：
- 追寻某味药材误入
- 妖兽暴动引发入口显现
- 主角在危急中闯入

### 3. 海底沉船遗迹群
**位置**：中海区域某处
**传闻**：
- 古代海上丝绸之路的沉没船队
- 载有大量宝物和功法
- 被海兽盘踞
- 水下环境复杂，探索难度大

**可能的剧情**：
- 商船打捞时发现线索
- 陈家组织水下探索
- 主角因特殊体质（混沌气）能在水下活动

## 六、经济与生活

### 主要产业
1. **海洋渔业**：最大产业，养活近半人口
2. **矿石开采**：庆家主导，是重要经济支柱
3. **山林采集**：药农、猎户、伐木工
4. **海上贸易**：陈家主导，与其他城池通商
5. **冒险者经济**：任务、装备、食宿、娱乐

### 坊市情况
- **中心坊市**：最大的交易市场，日常开放
- **夜市**：晚间开放，有黑市混杂其中
- **拍卖行**：每月一次小型拍卖，由天渊商会分号承办
- **散修市场**：城东角落，散修摆地摊

### 生活成本
- **凡人家庭**：月开销约100-500文灵币
- **低阶修士**：月开销约1-5块下品灵石
- **中阶修士**：月开销约10-50块下品灵石
- **高阶修士**：不计其数

---

## 七、势力关系

### 基本格局
- **城主府**：名义上的管理者，居中制衡，背靠沧澜剑宗，是城中最高修仙战力所在
- **陈家**：有钱有势，控制海洋资源，作风霸道，一心压过庆家独大
- **庆家**：稳扎稳打，控制山林矿脉，作风内敛，是陈家最大的绊脚石
- **佣兵团**：多为两大家族与城主府的打手或附庸，也常卷入私斗

### 势力矛盾
1. **陈庆资源角力（明争暗斗的核心）**：陈家掌海贸、庆家掌矿脉，两族为码头经营权、矿税分成、城主府宠信乃至海兽潮后的物资配给常年角力。但两族从不爆发"家族战"——修仙界拳头大者说了算，三方都知私斗流血不划算，真正的资源归属由**三方比武**一锤定音（见下节）；日常里陈家仗财力与铁鹰团打手屡屡直接动手或暗手挤压庆家，庆家则凭矿脉根基与口碑苦撑、暗中结交丹师等新锐。
2. **散修与家族**：散修依附家族谋生，又常与家族争利，关系微妙
3. **城主与各方**：城主府既是比武主办也是参赛方，刻意维持陈庆均势、坐收渔利，两族都猜不透周沧海到底帮谁
4. **佣兵团内斗**：铁鹰团仗陈家之势欺压其他团，野草团等底层团受压最甚

### 海岸线资源比武（三方定期武比）
- **性质**：渔阳城本地资源分配武比（属"本地小规模武比"，非榜单赛事，只攒声望、定资源份额，不登任何正式榜单）。由城主府牵头兴办，是城内三股力量和平重划海岸线资源份额的法定机制，替代了年年混战的私斗。
- **周期**：每三年一届，今年恰逢比武之年，且比武落点紧贴海兽潮淡季后的资源盘点，海岸线资源（近海渔获、海产贸易权、近海灵脉采补权）是城池命脉，牵动全城利害。
- **参赛三方**：陈家、庆家、城主府。
- **规则**：三方各出**三名武者**，按总战绩排名，依次分得当年海岸线资源的**五成、三成、两成（名次→5/3/2）**。比武场上死伤各凭实力，城主府只做裁判与资源交割，不介入比武本身。
- **底层逻辑**：拳头大者说了算，三方不愿年年私斗流血，遂以比武代混战、名次定份额。城主府既办赛也下场——它本就统管海产资源收集，借比武把官方那一份合法化；周沧海深谙制衡，历来不让城主府独占头名（否则陈庆必联手反噬），多卡中游，把前两成大头留给陈庆争抢，以此维持均势。
- **与主角关联**：本届比武前，庆家缺一名够分的筑基战力，看中陆渊的筑基中期修为与丹师身价，将第三战席许给他；陈家（尤以陈玉之恨为引）不愿庆家凭此多占资源、更不愿陆渊坐大，遂借铁鹰团施压逼其退席。陆渊接席，成为庆家第三战力，与陈家正面对撞——三方比武由此成为渔阳城篇前期最硬的冲突主轴，替代了生硬的"家族战"。

### 外部威胁
- **海兽潮**：不定期爆发，是全城共同敌人
- **海盗**：远海活动，偶尔劫掠商船
- **散修盗匪**：山林中活动，打劫落单者

### 丹道经济与炼丹铺（一丹难求·丹术分层·委托规则）
- **一丹难求**：渔阳城临海尚武、冒险者鱼龙混杂，伤亡频仍，对疗伤丹、清灵丹需求极大；但城中无像样丹师，丹药全靠外地商队流入，量少价高，低阶疗伤丹都被炒到离谱，寻常修士受伤常只能硬扛。这是主角以丹道立足的最大红利窗口。
- **丹术分层（陆渊的炼丹术等级）**：炼丹一道分多层，层越高成丹率、丹药品质、控火精度越高。当前进度：一品（第6章3颗下品聚气丹）→ 二品（第9章二品极品清灵丹，练气7层=二品丹师）→ 三品（第12章混沌异火）。丹术层是"手艺等级"，与灵海节点"算力"互补：节点管推演/药性辨别，丹术层管手上底子。灵海大模型在联网命名（第8章）前处于"建网/成型"期，不可用于丹方推演；联网后才逐步成为丹道辅助。
- **炼丹铺委托规则**：第10章陆渊在坊市开炼丹铺（二品丹师阶段），不走"自己采药炼了卖"的低级路子，而是**接受药草委托**——委托人自备药材，他代为炼制，约定"**两份药材出三至五颗丹药**"（视药材品级与成丹率浮动），多余归己或留作抽成；不赊账、不议价、急单加钱。一头供货、一头炼制，丹师不差钱、不沾采药苦力，专心涨丹术与实力。
- **与主角人设**：一个丹师不差钱、不怂。陆渊凭丹术与筑基实力在城中直起腰杆，接单打脸质疑者，不靠野草团庇护、不扮可怜。

## 九、主角发展路径

### 阶段一：野草团底层（练气2-3层，第5-7章）
- 跟着野草团接低级任务
- 适应修仙世界，建立人脉
- 灵海缓慢变化，自己摸不清状况
- 与铁鹰团产生矛盾

### 阶段二：崭露头角（练气3-5层，第8-12章）
- 通过某次任务展现价值（危险感知、恢复快）
- 被某个势力注意到（陈家/庆家）
- 参与中等难度任务（遗迹探索、矿难救援等）
- 获得更多资源，修为提升
- 灵海变化更明显，开始有意识地研究

### 阶段三：散修丹师路线（练气5-7层，第13-20章）
- 主角走散修丹师+佣兵团路线，不加入任何本地宗门
- 系统自学功法，借混沌塔/灵海大模型推演突破
- 在散修与佣兵团中竞争成长
- 灵海探索进入新阶段（联网尝试）
- 筑基准备

### 阶段四：离开新手村（筑基期）
- 筑基成功
- 外出历练
- 前往更大的城市（如东江城）
- 接触更广阔的世界
