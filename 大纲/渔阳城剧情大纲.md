## 二、近期主线重心（第23-50章）

**主重心：筑基后回渔阳城，卷入海兽潮危机，顺势撕开秦家暗线。**

逻辑自洽：第23章结尾陆渊神识捕到海兽潮前兆，自然动机是"回城备战、保护佣兵团"。海兽潮是前期最重磅的大事件，筑基后的战力正好在此兑现爽点；秦家暗线作为化神层面的远期伏笔，待陆渊修为足够后再展开。

三线并行推进：
1. **实力线（丹道为主、战力为辅）**：筑基初期→筑基中期/后期；灵海节点180→360（解锁四品丹方推演，**节点=丹道算力**，详见《灵海大模型》"丹道辅助"）；战技成熟（流云步+潮汐三式+沧澜剑诀+魂针术类）；灵海大模型持续进化、解锁新模块
2. **身份线（丹道核心）**：二品丹师→三品丹师→海兽潮供丹功臣→**地丹榜候选（东域丹师圈初识）**→以"丹道新星"身份被各方拉拢/忌惮，从"散修客卿"跃迁为"渔阳城新贵"。（丹道是主角最快登顶的路线，详见《榜单与赛事体系》）
3. **暗线**：秦家私通海兽潮的证据逐渐浮出，陆渊从被动卷入到主动追查，与化神期势力初次交锋

## 三、阶段规划

### 阶段A：回城与暗流（第24-28章）
- 第24章：回渔阳城。海兽潮预警已起，全城紧张。陈家二公子陈玉盯上陆渊（拍卖会风头+二品丹师），想暗算/拉拢。陆渊以筑基初期实力+丹师身份周旋（玄阳真人、庆家庆兰、城主府赵刚）。爽点：筑基实力展示（剑气外放震慑）。钩子：海兽潮具体时间敲定，陈玉准备趁乱动手。
- 第25章：海兽潮前夜。陆渊大量炼疗伤丹/清灵丹（用药谷灵药+万药园），展现丹道底蕴。白明预警：玄阳真人将以"传功/论丹"名义图谋他（觊觎混沌异火/灵海）。爽点：丹师产能碾压。钩子：玄阳真人对陆渊的"异常"起疑。
- 第26章：备战中。**（注：原"守城/剑气外放清场"作废；本章重写为天星石洞天十五天备战——备丹/淬体/壮魂/磨招一体。陆渊筑基中期破境，灵海大模型扩节点、神识远了一大截。镜湖盘点越阶战力，压赵刚一线。出塔拒签征召令）** 秦家影子：与渔阳城海兽潮无关（按作者指令 21-23 章已拆开）。
- 第27章：出关。陆渊领三人出天星石洞天。**重点立佣兵团三人形象——大壮（练气九层，团里憨厚体修/得力兄弟）、阿福（练气七层+流云步小成，贱嘴装逼担当）、林轻语（安静如刀）**。陆渊赠大壮三品长刀、制定"不守城守人"方针。**收束：院子里天擦黑，四人围石桌坐，明日事明日再算**（**无钩子**，按作者指令"结尾不是一定要钩子"）。
- 第28章：城里风波（重制v5）。陆渊带野草团结阳天宗客卿令进**醉仙楼旁听**（非路上溜达）。**东江商队/外地来客在酒楼闲聊，只露"东江城丹道大会"一个悬念点**（不抖完整榜单设定，留后续空间）；东江商队管事老周主动敬酒递路子（"去东江城来隆兴商行货栈寻我"），**商队真正发挥作用**，主角旁听记下，顺带掂量自身丹道之路（渔阳城二品丹够用、出了地界不算稀奇）。**冲突设计修正**：酒楼内仅作 verbal 对峙（校尉上楼递征召令、陆渊贱嘴接招"赵统领这么看得起我？派个校尉亲自来请"），**两个筑基在酒楼里干仗不合理，真打搬至醉仙楼外街面**。校尉（筑基初期）**真打不怂**：拔刀抢攻、连环刀法、催刀势硬拼，是被真正打服而非"亮修为吓退"；陆渊（筑基中期）剑气层层格挡、灵力压上、由散化凝破刀势制喉，校尉刀没松人没跪、打输不怂，撂"这事没完"带人退（去搬救兵/复命），城防营硬气未散。赵刚不出场长谈。**陈玉退场**（不出现，仅陈家管事远远低头避开）。神识仅作探查感知（不写"三百丈"量化数字），战斗用剑气。**删"灵海深处那团异火安静烧着"那句没头没尾的抖机灵**（本章不提异火）；**删沧澜剑宗硬扯**（几缕剑气非招牌武技，不尬连），旁观者只评"筑基中期、有点门道"。**悬念不写死**：不写"丹供人不供"。客卿令仅作身份背景（不当前筹码）。**收束不灌水思考/感慨：冲突完回院直接盘膝调息、取丹炉炼清灵丹继续修炼**（无强制钩子，剧情靠行动推进）。

### 阶段B：高潮与跃迁（第29-33章，原28-33顺延）
- 第29章（原28）：海兽潮高潮。城防军统领赵刚（筑基中期）亲眼见陆渊价值，陆渊立大功。**供丹救城（海量疗伤丹/清灵丹，灵海大模型辅助控火量产）+ 二品丹师官方认证（筑基期更稳，官方背书）**——丹道能力首次大规模兑现，被东江城丹阁莫老记名举荐入**地丹榜**候选。身份跃迁。钩子：秦家内应身份逼近暴露。
- 第30章（原29）：战后追查。陆渊顺线索查出渔阳城某势力里有秦家内应（可设陈家与秦家暗通，或某散修）。与化神期势力（秦家）间接交锋。爽点：打脸内应+情报碾压。钩子：秦万山注意到陆渊（剑无痕之死/异火）。
- 第31章（原30）：各方重新洗牌，陆渊成渔阳城新贵。玄阳真人图穷匕见（想夺混沌塔/异火），陆渊识破并反制/脱身。大爽点：识破阴谋、反将一军。钩子：离开渔阳城铺垫（去更大舞台查家仇/天渊）。
- 第32-33章（原31-33）：收束渔阳城线。陆渊巩固筑基中期，药谷/万药园产出滚雪球，与庆家/城主府结善缘，与陈家/玄阳真人结死仇。为离开渔阳城做资源与情报准备。

### 阶段C：离开与更大世界（第34-50章，概述）
- **主动走出新手村**：不再被动卷入，而是借**正统赛事**进入大陆竞争圈。离开渔阳城后，前往**东江城**（东域区域中心城池，第一落脚点）：
  - 参与**东域丹盟大会**（每三年）→ 冲**地丹榜**前列 + **丹道新锐榜**榜首（丹道核心线崛起，灵海大模型辅助炼丹降维）
  - 参与**东域天骄会武**（每三年）→ 冲**东域天骄榜** / **地榜**（战力线起步）
- 远期路径：东江城 → **天丹城**（天元城市群中立枢纽·万药大会会场；北域天丹宗为丹道圣地、主导天丹榜）→ 冲**天丹榜**（丹道登顶大陆级，与天丹宗嫡传正面对决）；或 **东荒城**（沧澜剑宗主城）→ 宗门线。最终 **天元主城**（宗门大比/天骄战 → 天榜/大陆天骄榜）。详见《榜单与赛事体系》"主角登榜路径"。
- 追查主线：①剑无痕之仇的秦家主线 ②自身混沌灵根变异 ③父亲陆天行血脉/大伯谋害真相 ④天渊封印/四至宝线索
- 节奏：每卷保留"少年想回家"底色，爽点升级，反派不降智（秦家会学聪明、请更强帮手）。丹道核心线贯穿，灵海大模型辅助炼丹是登榜降维武器。

## 四、第24章设计（已写：24-东江来客）

**标题**：`24-东江来客`（原"回城/暗流·海兽潮"方案作废，按作者指示重设计；区域大城定名东江城，与沧澜剑宗主城东荒城区分）

**设计原则**：不急于展开海兽潮（仅作背景暗涌），注入新势力为后续"往东江城(区域中心城池)落脚、进而东荒城(宗门主城)/加入宗门"铺路。

**剧情梗概**：
- 陆渊筑基后回渔阳城，发现外出筑基期间城里来了两股东江城来客（**东江城 = 东域区域中心城池，主角走出渔阳城后的第一落脚点**；**东荒城另为沧澜剑宗主城，剑道圣地，二城不可混**）：
  1. **东江城商队**——车马规规矩矩进**城西货栈**，与聚宝阁做批发清点（玄铁、灵蚕绫、灵米），不街头摆摊叫卖；管事闲话透出"东荒城那边海兽近来不太平"，暗扣海兽潮主线。让陆渊第一次真切了解"往外走"的目标与格局
  2. **东江城丹阁莫老**（**筑基初期执事**，修为立得住）来寻海里特有药材"瑚月草"（生阴寒暗流、十年一株、炼凝魂丹续魂之主药），开价十块上品灵石/株，在聚宝阁留悬赏、由沈清秋接待
- **不写"宗门弟子历练"线**：沧澜剑宗是大陆顶级宗门（三大宗门之一，东域·沧澜海岸剑道霸主），其主城即东荒城，不可能派练气期弟子到渔阳城这等边远小城摆摊；强行凑此线既违设定又掉格，已删。陆渊与沧澜剑宗的渊源改由**地理+遗物**悄悄埋线——东荒城即宗门主城，他兜里那卷《沧澜剑诀》、那柄三品沧澜剑皆出自剑无痕（沧澜剑宗外门长老）残魂所赠，等于早牵上了宗门线，无需弟子到场
- 四条线织入：淬肉身(雷淬玉色底子/肉身反撞无赖立威)、打磨神魂(神识300丈隔空识破坊市两处以次充好骗局：凡草染灵、丹渣充好丹)、喂灵海大模型见识(180灵海大模型推演东江城(区域中心城池)与沧澜剑宗(主城东荒城)关系+翻车"米铺地窖"；并推演陈玉恶意)、提炼丹术(已是三品丹师，底牌不亮，对外只露二品丹识，点破瑚月草特征得莫老引荐)
- 爽点：筑基肉身反撞立威（无赖一掌被玉色肉身震退）+ 神识三百丈识破骗局显丹道眼光 + 二品丹识点破瑚月草得东江城丹阁引荐；沈清秋热切结交
- 钩子：莫老递"去东江城丹阁寻我"引荐（东江城=东域区域中心城池/落脚点，进阶方为东荒城=沧澜剑宗主城）= 通东江城与宗门之路；陈玉暗嫉（"等机会到了谁都保不了他"）+ 灵海大模型推演其迟早动手；海兽潮暗涌（商队管事闲话"东荒城海兽不太平"）仅背景一笔

**自检**：推进（境界筑基、情报东江城=东域区域中心城池/东荒城=宗门主城、关系丹阁引荐）、爽点（肉身立威+神识识破骗局+丹识得引荐）、钩子（东江城门路+宗门seed+陈玉+海兽潮）、不虐主、反派有逻辑（陈玉嫉恨）、与主线关联（东江城/宗门线开启，海兽潮与秦家暗线延后）。

**设定确认（与《世界地图》《势力架构》《渔阳城设定》一致）**：
- 沧澜剑宗 = 顶级势力·三大宗门之一（东域·沧澜海岸，剑道霸主）；**东荒城即其主城（剑道圣地）**；**东江城 = 东域区域中心城池（林家坐镇），主角走出渔阳城后的第一落脚点，与东荒城不可混**
- 渔阳城 = 沧澜剑宗辖下最北端的边远小城；本地宗门是**阳天宗**（三流末流，玄阳真人创派），**非**沧澜剑宗
- 剑无痕 = 沧澜剑宗外门长老（元婴期残魂），赠陆渊《沧澜剑诀》与三品沧澜剑——这是陆渊与顶级宗门的唯一渊源
- 切记：勿再将"沧澜剑宗"降格为三流/本地宗门，二者不可混淆（详见《常见错误记录》）

**注**：因本张重设计，原阶段A"第24章海兽潮预警"顺延——海兽潮爆发推至第25章之后，第24章改为"东荒来客"铺垫外出之路。

## 六、写作纪律（复述关键项，详见《行文规范.md》）

- 单章 2500-3500 字，1场景为主+1小冲突/信息揭露+结尾钩子
- 每章至少1小爽点，每5-8章1大爽点
- 钩子必须近两章内兑现，不放空炮
- 不写纯虐主、不写降智反派、不写后宫、不写注水
- 程序员梗/塔灵/大模型互动控量，大模型要有翻车
- 突破后必稳固修为，灵海节点/神识同步涨
- 主角面对撩拨/挑衅/煽情必须接住+反将，不脸红后退
