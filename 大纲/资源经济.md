# 资源经济体系

## 货币体系

### 灵石

**品级**：下品 / 中品 / 上品 / 极品 / 仙灵石

**兑换比例**：
- 1中品灵石 = 100下品灵石
- 1上品灵石 = 100中品灵石 = 10000下品灵石
- 1极品灵石 = 100上品灵石 = 1000000下品灵石
- 1仙灵石 = 100极品灵石 = 100000000下品灵石

**兑换规则**：
- 高阶灵石可向下兑换，但损失10%（如1中品兑换99下品）
- 低阶灵石不可向上兑换，需通过市场交易或宗门兑换
- 仙灵石极其稀有，仅在化神期以上交易中出现

**购买力示例**：
- 下品灵石：
  - 1块：普通外门弟子一天修炼所需
  - 5块：一顿像样的灵食
  - 10块：一件凡级下阶法器
  - 50块：一株普通灵草（如凝气草）
  - 100块：一瓶淬体液
- 中品灵石：
  - 1块：筑基期弟子一天修炼所需
  - 10块：一件凡级上阶法器
  - 50块：一瓶筑基丹（普通品质）
  - 100块：一株地级灵药
- 上品灵石：
  - 1块：金丹期弟子一天修炼所需
  - 10块：一件地级下阶法器
  - 50块：一瓶金丹丹
  - 100块：一株地级上品灵药
- 极品灵石：
  - 1块：元婴期弟子一天修炼所需
  - 10块：一件地级上阶法器
  - 50块：一瓶元婴丹
- 仙灵石：
  - 1块：化神期以上修士修炼所需
  - 仅在顶级交易中使用

### 灵币

**定义**：凡人世界流通的货币，由修仙者制造，蕴含微弱灵气

**面值**：1文 / 10文 / 100文 / 1两（1000文）

**兑换比例**：1000两灵币 ≈ 1下品灵石

**用途**：
- 凡人世界交易
- 凡人与低阶修士交易
- 日常生活开销

### 宗门贡献点

**定义**：各大宗门内部流通的虚拟货币

**获取途径**：
- 完成宗门任务
- 贡献资源（药材、矿石、法宝等）
- 宗门大比奖励
- 特殊贡献

**用途**：
- 兑换功法秘籍
- 兑换修炼资源（灵石、丹药、材料）
- 兑换修炼场地使用权
- 兑换宗门福利

**特点**：
- 仅限本宗门使用，不可跨宗门兑换
- 每月清空部分（防止囤积）
- 贡献点等级与宗门地位挂钩

### 特殊代币

**炼丹师工会积分**：
- 通过完成炼丹任务、上交丹药获取
- 可兑换丹方、药材、丹炉
- 与炼丹师等级挂钩

**炼器师工会积分**：
- 通过完成炼器任务、上交法宝获取
- 可兑换炼器图谱、矿石、炼器工具
- 与炼器师等级挂钩

## 交易场所

### 坊市

**开放时间**：每日辰时到亥时

**交易规则**：
- 自由交易，双方议价
- 需缴纳5%交易税（由管理者收取）
- 禁止强买强卖
- 禁止交易违禁品

**常见商品**：
- 灵石、丹药、法器
- 灵草、矿石、妖兽材料
- 功法秘籍（低级）、符箓、阵盘
- 灵食、灵酒、日用品

**规模等级**：
- **小城坊市**：渔阳城等，交易以凡级物品为主
- **区域坊市**：梅城、东江城等，交易以地级物品为主
- **核心坊市**：天元主城、丹城、天工城，交易涵盖所有品级

### 拍卖会

**举办频率**：
- 小型拍卖会：每月一次
- 中型拍卖会：每季一次
- 大型拍卖会：每年一次
- 特级拍卖会：每十年一次（天元主城聚宝阁）

**参与门槛**：
- 小型：无门槛，自由参与
- 中型：需缴纳10块中品灵石保证金
- 大型：需持有天渊商会会员令牌
- 特级：需化神期以上修为或缴纳100块上品灵石保证金

**拍品等级**：
- 小型：凡级物品为主，偶尔地级物品
- 中型：地级物品为主，偶尔地级上品
- 大型：地级上品为主，偶尔极品物品
- 特级：极品物品为主，可能出现天材地宝

**规则**：
- 价高者得
- 需缴纳10%佣金
- 拍品真假自负
- 可匿名竞拍（需额外缴纳保证金）

### 黑市

**位置**：各大城池隐蔽角落，通常在地下

**特殊规则**：
- 无监管，风险自负
- 交易税高达20%
- 可交易违禁品
- 无保障，可能遭遇抢劫

**禁忌物品**：
- 天渊至宝相关物品
- 各宗门镇宗功法
- 渡劫期强者遗物
- 噬魂殿相关物品

**常见商品**：
- 稀有材料、违禁法器
- 被盗物品、赃物
- 禁术秘籍、邪恶功法
- 特殊情报、暗杀服务

## 资源产出

### 灵石矿脉

**分布**：
- **南域铁山平原**：大陆最大灵石矿脉，由百炼宗掌控
- **东域沧澜海岸**：海底矿脉，品质较高但开采困难
- **北域万药山脉**：零散矿脉，多为低品灵石
- **天元城周边**：中等矿脉，由大炎王朝管理

**开采权**：
- 大型矿脉：由顶级势力掌控（百炼宗、三大宗门）
- 中型矿脉：由一流势力或二流势力掌控
- 小型矿脉：由三流势力或散修团体开采

**争夺情况**：
- 矿脉是各大势力的核心资源
- 矿脉争夺时常引发战争
- 新矿脉发现会引发新一轮争夺

**品级分布**：
- 下品灵石矿：占总量70%
- 中品灵石矿：占总量20%
- 上品灵石矿：占总量8%
- 极品灵石矿：占总量1.5%
- 仙灵石矿：极其稀有，仅传说中存在

### 灵药园

**人工种植**：
- 各宗门、家族均有灵药园
- 天丹宗百草园规模最大、品种最全
- 药仙谷有独特培育方法，部分药材为谷中独有

**野外采集**：
- 万药山脉是主要采集地
- 危险区域药材品质更高
- 需组队前往，防范妖兽

**珍稀品种**：
- **地级灵药**：千年人参、万年灵芝、悟道花等
- **天级灵药**：混沌灵果、涅槃花、时空草等（极其稀有）
- **特殊灵药**：需特殊环境生长（如极寒、极热之地）

### 矿物资源

**炼器材料**：
- 铁精、铜精、银精、金精（凡级）
- 寒铁、精钢、玄铁、陨铁（地级）
- 仙金、神铁、混沌石（天级，极其稀有）

**炼丹材料**：
- 灵草、灵花、灵果、灵液
- 妖兽材料（兽丹、兽皮、兽骨等）
- 特殊材料（灵火、灵水、天地精华等）

**特殊矿物**：
- 空间石：用于炼制空间法器
- 魂石：用于炼制魂器
- 法则结晶：蕴含法则之力的矿石

## 资源分配

### 宗门分配

**内门弟子**：
- 每月发放：10块中品灵石 + 基础丹药
- 可申请修炼资源（需贡献点）
- 可进入宗门秘境修炼

**外门弟子**：
- 每月发放：5块下品灵石 + 基础修炼材料
- 需完成任务获取额外资源
- 可参与宗门修炼课程

**杂役弟子**：
- 每月发放：2块下品灵石
- 可通过劳动换取资源
- 表现优秀可晋升外门

### 家族分配

**嫡系**：
- 核心资源优先分配
- 可获得家族传承功法
- 可使用家族修炼场地

**旁系**：
- 按贡献分配资源
- 可获得部分传承功法
- 资源有限，需竞争获取

**外姓**：
- 仅能获取基础资源
- 无法获得核心传承
- 需通过联姻或功绩提升地位

### 市场交易

**价格波动**：
- 受供需关系影响
- 受季节影响（如药材采摘季节价格下降）
- 受战争影响（战时灵石价格上涨）
- 受秘境开启影响（秘境开启后稀有物品增多，价格下降）

**供需关系**：
- 稀有物品：供给少，价格高
- 常用物品：供给稳定，价格波动小
- 战略资源：受势力掌控，价格受控

## 特殊资源

### 天材地宝

**定义**：天地自然孕育的顶级资源，蕴含浓郁灵气或法则之力

**等级**：
- 地级天材地宝：如千年人参、万年灵芝、悟道花
- 天级天材地宝：如混沌灵果、涅槃花、时空草
- 传说级：如凤凰蛋、龙鳞、混沌本源（仅传说）

**用途**：
- 炼制顶级丹药
- 锻造顶级法宝
- 辅助突破境界
- 提升修炼资质

**获取途径**：
- 秘境探索
- 远古遗迹
- 天地奇险之地
- 拍卖会竞拍

### 悟道资源

**定义**：可辅助修士领悟法则、突破瓶颈的特殊资源

**种类**：
- **悟道茶**：饮用后可静心悟道，提升领悟效率
- **悟道石**：蕴含法则感悟，可辅助领悟法则
- **悟道树**：树下修炼可提升悟道效率
- **法则碎片**：蕴含法则之力的碎片

**效果**：
- 缩短悟道时间
- 提升领悟成功率
- 辅助突破瓶颈

**稀有度**：极其稀有，仅顶级势力拥有少量

### 突破资源

**定义**：帮助修士突破境界的特殊资源

**种类**：
- **筑基丹**：帮助练气期突破筑基期
- **金丹丹**：帮助筑基期突破金丹期
- **元婴丹**：帮助金丹期突破元婴期
- **化神丹**：帮助元婴期突破化神期
- **炼虚丹**：帮助化神期突破炼虚期
- **合体丹**：帮助炼虚期突破合体期
- **渡劫丹**：传说中帮助合体期突破渡劫期的丹药

**效果**：
- 提升突破成功率
- 减少突破风险
- 提升突破后修为基础

**限制**：
- 每颗丹药仅能使用一次
- 多次使用效果递减
- 品质越高效果越好

## 资源获取途径

### 宗门任务

**类型**：
- 巡逻任务：维护区域安全
- 采集任务：采集灵草、矿石
- 猎杀任务：猎杀妖兽获取材料
- 护送任务：护送重要人物或物资
- 探索任务：探索未知区域

**奖励**：
- 灵石、贡献点
- 丹药、材料
- 功法、武技

### 历练探索

**方式**：
- 组队探索秘境
- 独自闯荡危险区域
- 寻找远古遗迹

**收获**：
- 稀有材料、天材地宝
- 功法传承、远古遗物
- 机缘奇遇

**风险**：
- 遭遇强大妖兽
- 遭遇敌对势力
- 迷失方向

### 秘境争夺

**秘境类型**：
- 万药秘境：生长珍稀古药
- 剑冢秘境：蕴含上古剑意
- 矿山秘境：有珍稀矿藏
- 天渊秘境：蕴含神秘传承

**争夺规则**：
- 需特定令牌或条件进入
- 秘境内部禁止大规模争斗（部分秘境）
- 秘境开启时间有限

### 炼制加工

**炼丹**：
- 将灵草炼制成丹药
- 炼丹师等级越高，成功率越高
- 高品质丹药价值远超原材料

**炼器**：
- 将矿石炼制成法器
- 炼器师等级越高，品质越好
- 高品质法器价值极高

**符箓**：
- 将灵力注入符箓，形成术法
- 符箓师等级越高，威力越强
- 高阶符箓可媲美地级术法

**阵法**：
- 布置阵法，可用于防御、攻击、辅助
- 阵法师等级越高，阵法越精妙
- 高阶阵法可困杀化神期强者

## 经济规则

### 通胀控制

**灵石供给**：
- 灵石矿脉产出稳定，不会突然增加
- 高阶灵石极其稀有，不会泛滥
- 仙灵石几乎绝迹，不会进入流通

**资源稀缺性**：
- 顶级资源（天材地宝、悟道资源）极其稀有
- 不会出现大量涌入市场的情况
- 供需关系保持平衡

**交易税**：
- 坊市交易税5%，拍卖会佣金10%，黑市20%
- 税费收入用于维护市场秩序和资源开发
- 防止投机倒把

### 价值体系

**价值锚点**：
- 下品灵石：基础价值单位
- 修炼资源：与修士需求挂钩
- 天材地宝：稀缺性决定价值

**价格稳定**：
- 常用物品价格波动小（±10%）
- 稀有物品价格波动大（±50%）
- 顶级物品价格由拍卖会决定

### 经济平衡

**三大宗门制衡**：
- 天丹宗掌控丹药供给
- 百炼宗掌控法宝和灵石供给
- 沧澜剑宗掌控武力保障
- 三者相互依赖，维持经济平衡

**商会调节**：
- 天渊商会通过贸易网络调节资源分配
- 拍卖会定期释放稀有资源
- 商会储备可应对资源短缺

**市场机制**：
- 自由市场调节价格
- 供需关系决定资源流向
- 势力垄断受市场制约
