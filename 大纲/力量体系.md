# 力量体系

## 法术体系

### 五行术法

**金系术法**
- 基础：金刃术、金盾术、金刺术
- 进阶：裂金斩、金光罩、金雷术
- 高阶：金法则·万刃齐发、金法则·金刚不坏
- 特点：攻击力强，锋利无比，擅长破甲
- 对应灵根：金系灵根

**木系术法**
- 基础：藤鞭术、木盾术、生长术
- 进阶：万木逢春、荆棘丛生、木之结界
- 高阶：木法则·生命汲取、木法则·万物复苏
- 特点：生命力强，擅长治疗和防御，可控制植物
- 对应灵根：木系灵根

**水系术法**
- 基础：水箭术、水盾术、水流术
- 进阶：海啸术、冰封术、水之牢笼
- 高阶：水法则·无尽汪洋、水法则·冰冻三尺
- 特点：适应性强，攻守兼备，可操控水流
- 对应灵根：水系灵根

**火系术法**
- 基础：火球术、火盾术、火焰术
- 进阶：烈焰风暴、焚天火焰、火之领域
- 高阶：火法则·涅槃之火、火法则·焚尽万物
- 特点：攻击力强，温度极高，擅长灼烧
- 对应灵根：火系灵根

**土系术法**
- 基础：土刺术、土盾术、岩石术
- 进阶：地裂术、石化术、土之牢笼
- 高阶：土法则·大地之力、土法则·山岳崩塌
- 特点：防御力强，厚重沉稳，可操控大地
- 对应灵根：土系灵根

### 特殊术法

**雷法**
- 基础：雷击术、雷盾术、雷网术
- 进阶：天雷术、雷龙术、雷之领域
- 高阶：雷法则·天罚之雷、雷法则·雷霆万钧
- 特点：攻击力极强，速度极快，附带麻痹效果
- 对应灵根：雷系变异灵根

**风法**
- 基础：风刃术、风盾术、御风术
- 进阶：风暴术、风之隐身、风之速度
- 高阶：风法则·空间切割、风法则·瞬息万里
- 特点：速度极快，擅长闪避和切割，可操控风之力
- 对应灵根：风系变异灵根

**冰法**
- 基础：冰箭术、冰盾术、冰冻术
- 进阶：冰风暴、冰之封印、冰之领域
- 高阶：冰法则·绝对零度、冰法则·冰封天地
- 特点：控制力强，擅长冰冻和封印，温度极低
- 对应灵根：冰系变异灵根

**毒法**
- 基础：毒雾术、毒刺术、毒刃术
- 进阶：万毒噬心、毒之领域、毒之诅咒
- 高阶：毒法则·万毒归宗、毒法则·腐蚀万物
- 特点：攻击力阴毒，擅长持续伤害和腐蚀，可操控毒素
- 对应灵根：毒系变异灵根

**暗法**
- 基础：暗影术、暗盾术、暗刺术
- 进阶：暗影潜行、暗之吞噬、暗之领域
- 高阶：暗法则·吞噬一切、暗法则·虚无缥缈
- 特点：擅长隐匿和偷袭，可操控黑暗之力
- 对应灵根：暗系变异灵根

**剑术**
- 基础：基础剑法、剑气术、御剑术
- 进阶：沧澜七剑、一剑破万法、潮汐剑法
- 高阶：剑道法则·万剑臣服、剑道法则·一剑斩天
- 特点：攻击力强，速度快，剑意可破万法
- 对应灵根：金系灵根（剑修专属）

**幻术**
- 基础：迷幻术、幻象术、催眠术
- 进阶：幻域、幻境迷宫、灵魂幻术
- 高阶：幻法则·真实幻境、幻法则·颠倒乾坤
- 特点：攻击无形，擅长精神控制和迷惑
- 对应灵根：暗系灵根或先天魂体

**魂术**
- 基础：灵魂感知、灵魂传音、灵魂冲击
- 进阶：灵魂禁锢、灵魂控制、灵魂分身
- 高阶：魂法则·灵魂掌控、魂法则·灵魂灭杀
- 特点：直接攻击灵魂，无视肉身防御
- 对应灵根：暗系灵根或先天魂体

| 法则 | 对应条件 | 核心能力 | 领悟难度 |
|------|---------|---------|---------|

| 造化法则 | 渡劫期 | 创造、演化、改变 | 传说 |
| 轮回法则 | 渡劫期 | 轮回、重生、因果 | 传说 |
| 天道法则 | 传说 | 掌控天地、改写规则 | 传说 |

### 法则领悟规则

1. **灵根匹配**：修士优先领悟与自身灵根属性相同的法则，灵根属性决定法则领悟方向
2. **灵根纯度**：灵根越纯净（单属性灵根），法则领悟速度越快，法则威力越强；杂灵根领悟速度慢，但可融合多种法则
3. **变异灵根**：变异灵根对应变异法则，变异法则威力更强但领悟难度更高，需特殊机缘
4. **悟性决定**：法则领悟深度取决于修士悟性，而非修为，悟性高者可越级领悟法则
5. **机缘重要**：高级法则和天地法则需要特殊机缘才能领悟，非灵根所能决定
6. **混沌灵根**：可领悟所有法则，领悟速度快，是最特殊的灵根，传说中可融合所有法则
7. **法则融合**：高阶修士可融合多种基础法则，形成独特法则，如金+雷=天罚法则
8. **属性克制**：法则之间存在克制关系，如火克木、水克火，克制法则可削弱被克制法则威力

## 法宝体系

### 法器品级总表

法器分一品~七品，对应练气期~渡劫期。各品法器的材质、特点、代表物品详见`修炼体系.md`"炼器师评级体系"。

| 品级 | 对应境界 | 核心特征 | 器灵 |
|------|---------|---------|------|
| 一品 | 练气期 | 蕴微弱灵气，比凡铁锋利数倍 | 无 |
| 二品 | 筑基期 | 可注灵气增强，简单属性加成，可认主 | 无 |
| 三品 | 金丹期 | 可刻画简单阵法，特殊效果（破甲/吸血） | 无 |
| 四品 | 元婴期 | 可刻画中等阵法，初步具备成长性 | 可孕育器灵，辅助战斗 |
| 五品 | 化神期 | 可刻画复杂阵法，有领域效果 | 器灵初具智慧，可自主护主 |
| 六品 | 炼虚期 | 可铭刻领域符文，蕴含领域之力 | 器灵完整智慧，可化形而出 |
| 七品 | 合体/渡劫期 | 可铭刻天地法则符文，蕴含天地之力 | 器灵可化人形，可成长为通天灵宝 |

> 备注：据传当世能炼制七品法器的炼器师不超过三人，大陆现存七品法器不超过十件。

### 法宝分类

**攻击类**：飞剑、刀斧、枪戟、锤棍、弓弩、锁链等
**防御类**：盾牌、铠甲、护符、宝塔、法衣等
**辅助类**：储物袋、飞行法器、阵盘、符箓、玉简等
**特殊类**：魂器（专克灵魂）、丹炉（辅助炼丹）、傀儡（可操控战斗）、阵旗（布置阵法）、罗盘（探查寻宝）等

### 本命法宝

- 修士以自身精血和灵魂之力祭炼的法宝，**一生只能祭炼一件**
- 与修士灵魂绑定，不可分离，随修士修为提升而成长
- 本命法宝被毁，修士会受重伤
- 因唯一性，修士选择本命法宝极为慎重，多为品级高、潜力大的法宝
- 主角陆渊的本命法宝为混沌塔（天渊至宝残片），无需另寻

丹药分一品~七品，对应练气期~渡劫期。各品丹药的效果、代表丹药详见`修炼体系.md`"炼丹师评级体系"与"丹药类型详细分类"。

| 品级 | 对应境界 | 核心特征 | 代表 |
|------|---------|---------|------|
| 一品 | 练气期 | 辅助基础修炼，药效温和 | 聚气丹、止血丹、清毒丹 |
| 二品 | 筑基期 | 加速修炼，辅助小瓶颈突破 | 筑基丹、疗伤丹、破障丹 |
| 三品 | 金丹期 | 大幅加速修炼，辅助大境界突破 | 金丹丹、回春丹、破厄丹 |
| 四品 | 元婴期 | 极强辅助，可提升资质/体质 | 元婴丹、洗髓丹、灵骨丹 |
| 五品 | 化神期 | 蕴含法则之力，可永久提升属性 | 化神丹、法则丹、悟道丹 |
| 六品 | 炼虚期 | 蕴含领域之力，助领悟法则 | 炼虚丹、领域丹、涅槃丹 |

> 备注：据传大陆现存七品丹药寥寥无几，炼制所需材料万年来未曾凑齐。

### 丹毒与抗药性

**丹毒**：
- 长期服用同一种丹药会积累丹毒
- 丹毒积累过多会影响修炼，甚至走火入魔
- 需服用解毒丹或通过修炼化解

**抗药性**：
- 多次服用同一种丹药会产生抗药性
- 抗药性会降低丹药效果
- 需换用更高品质或不同种类的丹药

## 符箓体系

### 符箓品级总表

符箓分一品~七品，对应练气期~渡劫期，为一次性战斗耗材。

| 品级 | 对应境界 | 核心特征 | 代表 |
|------|---------|---------|------|
| 一品 | 练气期 | 基础攻击/防御，威力较弱 | 火球符、护盾符 |
| 二品 | 筑基期 | 中等攻击/防御，可增幅术法 | 雷符、烈焰符、铁甲符 |
| 三品 | 金丹期 | 强力攻击/防御，可释放完整术法 | 爆破符、冰封符、金刚符 |
| 四品 | 元婴期 | 极强攻击/防御，可自动追踪目标 | 天雷符、地火符、万法符 |
| 五品 | 化神期 | 蕴含法则之力，可造成法则级伤害 | 法则符、空间符、灵魂符 |
| 六品 | 炼虚期 | 蕴含领域之力，可开辟小型领域 | 领域符、时间符、封印符 |
| 七品 | 合体/渡劫期 | 蕴含天地法则，可改写局部规则 | 天地符、造化符、轮回符 |

### 符箓分类

**攻击类**：火球符、雷符、爆破符、天雷符、法则符
**防御类**：护盾符、铁甲符、金刚符、万法符、封印符
**辅助类**：御风符、隐身符、瞬移符、追踪符、时间符
**特殊类**：灵魂符、诅咒符、召唤符、造化符、轮回符

### 符箓师地位

- 符箓师**不是主流修炼方向**，各宗门有专门配置，但从未成为修炼主流
- 符箓本质是**战斗耗材**，一次性消耗品，使用后即失效
- 符箓师通常由其他修炼方向的修士兼职，而非专职修炼符箓
- 高阶符箓价值极高，是**战略性物资**，各大势力争相储备
- 符箓师数量稀少，但地位特殊，更多被视为工匠而非修士
- 战斗中符箓是**消耗品**，如同箭矢弹药，用完即弃

## 阵法体系

### 阵法品级总表

阵法分一品~七品，对应练气期~渡劫期。阵法为预设工事，布阵耗时较长，非即时战力。

| 品级 | 对应境界 | 核心特征 | 代表 |
|------|---------|---------|------|
| 一品 | 练气期 | 基础聚灵、迷惑、简单防御 | 聚灵阵、迷魂阵 |
| 二品 | 筑基期 | 中等攻击、防御、困敌 | 攻击阵、防御阵、困兽阵 |
| 三品 | 金丹期 | 强力攻击、防御、困杀 | 杀阵、迷阵、幻阵、五行阵 |
| 四品 | 元婴期 | 极强攻防困杀，可困住元婴期修士 | 四象阵、八卦阵、万剑阵 |
| 五品 | 化神期 | 法则级攻防困杀，可困杀化神期修士 | 法则阵、领域阵、诛神阵 |
| 六品 | 炼虚期 | 领域级攻防困杀，可开辟独立空间 | 空间阵、时间阵、轮回阵、封天阵 |
| 七品 | 合体/渡劫期 | 天地级攻防困杀，可改写局部规则 | 天地阵、造化阵、混沌阵、飞升阵 |

### 阵法分类

**聚灵类**：聚灵阵、灵脉阵、洞天阵
**攻击类**：杀阵、诛神阵、灭魂阵、万剑阵
**防御类**：护阵、封天阵、金刚阵、万法阵
**困敌类**：困阵、迷阵、幻阵、空间阵
**特殊类**：传送阵、时间阵、轮回阵、造化阵

### 阵法师地位

- 阵法师**不是主流修炼方向**，各宗门有专门配置，但从未成为修炼主流
- 阵法主要用于**宗门防御**和**秘境探索**，非战斗主流手段
- 阵法师通常由其他修炼方向的修士兼职，阵法只是其职业技艺
- 高阶阵法师极其稀少，是各大势力**争抢的战略性人才**
- 阵法是**战略性防御手段**，而非进攻手段，如同城池堡垒
- 战斗中阵法更多是**预设工事**，而非即时战力，布阵耗时较长

**等级**：一品~七品，对应练气期~渡劫期
**特点**：炼丹师品级仅代表技艺水平，不代表战斗实力

## 三大主流修炼方向

### 剑修

**核心能力**：剑气外放、剑意领悟、御剑飞行、人剑合一
**攻击特点**：攻击力强，速度快，擅长远程攻击和点杀
**防御特点**：防御较弱，需靠身法闪避
**代表功法**：沧澜剑宗《沧澜剑诀》、飘渺剑派《飘渺剑法》
**对应灵根**：金系灵根为主

### 体修

**核心能力**：肉身强悍、力量巨大、恢复能力强、防御惊人
**攻击特点**：近战无敌，爆发力强，擅长以力破巧
**防御特点**：防御极强，可硬抗法宝攻击
**代表功法**：百炼宗《铁骨金身诀》、李家《金刚不坏神功》
**对应灵根**：土系灵根为主

### 魂修

**核心能力**：灵魂感知、灵魂攻击、灵魂控制、灵魂防御
**攻击特点**：攻击无形，难以防御，擅长精神控制
**防御特点**：肉身较弱，需保护自身
**代表功法**：天丹宗《炼魂经》、散修联盟《魂天诀》
**对应灵根**：暗系灵根或先天魂体
