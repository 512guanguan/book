# 行文规范

> 本文档用于指导后续小说写作，规范行文基调、情节设计、节奏推进、对话风格等，确保全书风格统一、爽感在线、幽默得当。

---

## 一、总体基调

### 核心原则

1. **积极向上为主色**：全书底色是少年的乐观与韧性，不是苦大仇深的复仇文。即便身处逆境，主角的内心戏也带着自嘲与幽默，让读者笑着看他翻盘
2. **幽默是底色而非点缀**：参考《修真聊天群》的日常吐槽、《亏成首富》的反套路喜剧感，幽默要融入叙述肌理，不是偶尔抖个机灵
3. **爽感是硬指标**：参考《斗破苍穹》"三十年河东三十年河西"的爽点密度，每个章节至少有一个让读者"噢"一声的小爽点，每5-8章一个大爽点
4. **装逼要有底气**：参考《逆天邪神》云澈的装逼——先藏拙，再一击反转，装逼前必有铺垫，装逼后必有旁观者震惊视角，绝不无脑装

### 情绪曲线

```
章节内：低谷→蓄力→小爽点收尾（单章自洽）
卷内：  平缓→冲突→高潮→余韵（卷有起承转合）
全书：  每卷爽点升级，每卷底色仍是"少年想回家"
```

**禁忌**：
- 不写纯虐主章节（主角可以吃亏，但当章或下章必有找回场子）
- 不写拖沓的内心纠结（纠结不超过半段，立刻转为行动）
- 不写苦情戏渲染（亲人牵挂用一两句带过，不煽情堆砌）

## 二、主角人设执行

### 陆渊的"幽默人设"落地

主角的幽默不是耍宝，而是**程序员思维碰撞修仙世界**产生的天然喜剧效果。核心来源：

1. **认知错位的吐槽**：用现代/程序员视角解读修仙常识，产生反差萌
   - 例：看到"打坐吐纳"，内心："这不就是 BIOSS 刷固件吗，灵气当数据流，丹田当硬盘……"
   - 例：第一次见飞剑，内心："好家伙，这不就是无人机？还是声控的。"

2. **自嘲式内心戏**：遇险不慌张，先在心里吐槽自己，再想对策
   - 例：被追杀时边跑边想："我一个 985 毕业的程序员，穿越来当杂役就算了，还要被堂兄追着砍，这简历没法写。"

3. **塔灵的冷笑话互动**：塔灵性格古朴沉稳却爱讲冷笑话，与主角形成"捧哏逗哏"组合
   - 塔灵讲冷笑话→主角无语→塔灵认真解释→主角更无语
   - 关键时刻塔灵一句正经话化解尴尬，或一句冷笑话缓解紧张

4. **灵海大模型的"AI式回答"**：大模型推理时偶尔给出过度字面化、过度严谨的回答，产生喜剧效果
   - 例：主角问"这功法怎么练最快"，大模型回复一串"参数调优建议"，主角："说人话。"大模型："多练。"

### 程序员思维的战斗应用

参考《亏成首富》的系统流爽感，让"程序员思维"在战斗中成为可见的优势：

1. **战斗前推演**：用灵海大模型模拟对手招式，找出最优解（爽点：对手以为自己必杀的一招，被主角精准破解）
2. **功法模块化拆解**：把复杂功法拆成"模块"理解，别人死记硬背，主角理解原理后举一反三（爽点：学功法速度碾压同辈）
3. **丹方/阵法优化**：用"流程优化"思路改进丹方，提升成功率或药效（爽点：炼出超品丹药震惊炼丹师）
4. **找Bug式破阵**：把阵法当程序找漏洞，专打阵眼（爽点：以低修为破高阶阵法）

> 注意：程序员思维是**放大器**不是**外挂**，它让主角学得快、想得深，但不能凭空产生实力。实力仍需修炼积累。

### 主角的"度"

- **不圣母**：善良有底线，对敌人不留情，对恶人不下不去手的犹豫不超过一句内心戏
- **不滥杀**：不主动欺凌弱小，但被惹了绝不忍气吞声
- **不后宫**：感情线单一（红颜知己），不搞暧昧流水账
- **不装弱过头**：藏拙是为了时机，不是怂到底，该亮剑时果断亮剑

## 三、故事情节设计

### 爽点设计公式

参考《斗破苍穹》的爽点结构，每个爽点应包含三要素：

1. **压抑蓄力**：先让主角或主角在乎的人被压制/羞辱/看轻（1-3章铺垫）
2. **反转爆发**：主角以出乎意料的方式翻盘（核心爽点，1章内完成）
3. **旁观震惊**：通过配角/敌人的视角渲染震惊，放大爽感（半章到1章）

**爽点类型库**（轮换使用，避免单一）：
| 类型 | 说明 | 适用阶段 |
|------|------|---------|
| 打脸装逼 | 被看轻后一击反转 | 全程 |
| 越级战斗 | 以低修为胜高修为 | 金丹期后 |
| 炼丹/炼器震惊 | 超品/异象震惊行家 | 筑基期后 |
| 身份揭露 | 隐藏实力/身份曝光 | 关键节点 |
| 智斗反杀 | 用计谋坑掉强敌 | 元婴期后 |
| 金手指新解锁 | 混沌塔新层/大模型升级 | 各阶段 |
| 旧账清算 | 复仇/打脸曾经欺负主角的人 | 各阶段 |

### 反派设计原则

参考《逆天邪神》——反派不蠢，才显得主角赢下来够爽：

1. **反派有逻辑**：反派行为有自洽的动机和利益考量，不是为坏而坏
2. **反派有成长**：主角变强，反派也在变强或请更强的帮手，不重复"主角升级回来碾压"的套路
3. **反派有记忆**：被主角坑过的反派会学聪明，不会反复送人头
4. **反派有人格**：即便是反派，也有可敬之处或可怜之处，不脸谱化
5. **大BOSS留悬念**：陆天霸、噬魂殿等大反派，前期只露冰山一角，逐步揭露其全貌

### 情节推进的"钩子"技巧

每章结尾必须留钩子，驱动读者翻下一章：

- **悬念钩子**：抛出新谜题（"塔灵突然说了一句奇怪的话……"）
- **冲突钩子**：预告即将到来的冲突（"明天就是宗门大比，陆渊看着名单，眼神一凝。"）
- **反转钩子**：当章结尾反转（"他以为事情结束了，却没注意到角落里那双眼睛。"）
- **爽点预告**：暗示下章爽点（"丹炉里传来异响，连丹塔长老都站了起来。"）

**禁忌**：
- 不用"欲知后事如何且听下回分解"式废话钩子
- 钩子必须在下一章或近两章内兑现，不放空炮

## 四、节奏控制

### 章节节奏

- **单章字数**：2500-3500字为宜，不拖沓
- **单章结构**：1个场景为主，1个小冲突或1个信息揭露，结尾留钩
- **信息密度**：每章必须推进至少一项（修为/情报/关系/爽点），不写纯水章

### 卷节奏

参考《斗破苍穹》的卷结构：

```
开卷（铺垫）：新环境/新角色/新目标引入，2-3章
发展（蓄力）：小冲突串联，主角暗中成长，5-8章
高潮（爆发）：大冲突+大爽点，2-3章
收尾（余韵）：清账+埋下一卷伏笔，1-2章
```

### 修炼与剧情的平衡

**不写纯修炼章**：修炼必须嵌在剧情中推进，参考以下方式：
- 修炼中穿插塔灵对话/大模型吐槽
- 修炼突破时安排外界干扰（敌人来袭/秘境开启）
- 修炼成果立刻用于下一次冲突
- 修炼瓶颈作为剧情驱动力（"必须找到某灵药才能突破"）

### 禁忌

- 不写大段功法原文背诵（最多两句口诀，重点写主角的理解和感悟）
- 不写流水账式日常（吃饭逛街除非有剧情作用，否则一笔带过）
- 不写重复的升级描写（同境界突破不重复渲染流程）

## 五、对话规范

### 对话原则

1. **每句对话要有信息量或性格**：不写"是啊""没错"这种填充对话
2. **对话推动剧情**：每段对话至少推进一项（情报/关系/冲突）
3. **对话区分人物**：不同人物说话方式不同
   - 陆渊：偶尔现代词汇，内心戏幽默，表面沉稳
   - 塔灵：古朴沉稳，偶尔冷笑话，对现代词汇好奇
   - 陆天霸：阴险虚伪，表面慈祥
   - 红颜知己：根据其身份设定语风（后续补充）
4. **少用对话标签**：能从语气判断说话人时，省略"他说""她道"

### 幽默对话模板

**模板1：认知错位吐槽**
```
配角（骄傲）："此乃我宗镇宗之宝《沧澜剑诀》，练至大成可剑斩天地！"
陆渊（内心）：剑斩天地……这不就是 root 权限吗？
陆渊（表面）："好功法，好功法。"
```

**模板2：塔灵冷笑话**
```
陆渊："塔灵，这关怎么过？"
塔灵："昔日主人曾言，遇事不决，量子力学。"
陆渊："……说人话。"
塔灵："打不过就跑。"
```

**模板3：大模型字面化回答**
```
陆渊："分析下这丹方的成功率。"
大模型（灵海浮现）：『经推演，当前丹方成功率约23.7%，主要瓶颈在于第三步火候控制，建议：1.将灵力输出降低12%……』
陆渊："直接告诉我能不能炼。"
大模型：『能。但会炸炉。』
```

**模板4：装逼后的淡定**
```
（主角一招秒杀强敌，众人震惊）
配角甲："这……这怎么可能！"
陆渊（收剑，语气平淡）："运气好。"
（内心）：大模型推演了八百遍，运气个鬼。
```

## 六、金手指使用规范

### 双金手指的分工

| 金手指 | 作用 | 限制 |
|--------|------|------|
| 混沌塔 | 修炼加速/储物/炼丹/锻体/魂修/秘藏 | 进入需安全环境；空间封锁无法进；塔身留原地有风险；第六层需天境开启 |
| 灵海大模型 | 快速推理/知识理解/学习加速/战斗推演 | 消耗神识；受限于已知信息；早期算力有限 |

### 金手指使用原则

1. **金手指放大努力，不替代努力**：主角仍需修炼、历练、冒险，金手指让他学得快、想得深，但不凭空给实力
2. **金手指要有代价**：用大模型消耗神识，用混沌塔需要安全环境，每次使用都要有成本考量
3. **金手指要有成长**：随修为提升，金手指能力逐步解锁，保持新鲜感
4. **金手指不能万能**：要设计金手指解决不了的场景（如感情、人心、未知领域的探索），迫使主角靠自身判断
5. **金手指要有"翻车"**：偶尔让大模型推演失误、混沌塔出意外，制造意外笑点或危机

### 禁忌

- 不写"主角开挂碾压一切"的无聊爽文
- 不写金手指万能到没有挑战
- 不写主角完全依赖金手指不思考

## 七、场景描写规范

### 描写原则

1. **少而精**：关键场景详写，过渡场景一笔带过
2. **感官为主**：多写视觉/听觉/触觉，少写抽象形容
3. **场景服务剧情**：描写要推动氛围或暗示信息，不堆砌辞藻
4. **战斗重节奏**：战斗描写重招式逻辑与节奏，不写慢动作堆砌

### 战斗描写模板

```
[起]：双方对峙，一句话点明实力差距（制造紧张感）
[承]：交手数招，主角处于下风或试探（蓄力）
[转]：主角使出底牌/金手指推演结果（反转）
[合]：一击决胜，旁观者震惊视角收尾（爽点）
```

**禁忌**：
- 不写"只见他使出一招XX，那招式如何如何"的解说式战斗
- 不写双方互相喊招式名的回合制战斗
- 不写超过两章的冗长大战

### 秘境/历练描写

参考《逆天邪神》的秘境设计：
1. **有谜题**：秘境不止打怪，要有机关/谜题/选择
2. **有取舍**：主角面临"拿这个还是那个"的选择，体现性格
3. **有意外**：秘境里总有计划外的变数，制造危机
4. **有收获也有代价**：不写无脑捡宝，每次收获都伴随风险或代价

## 八、伏笔与回收

### 伏笔原则

1. **埋得自然**：伏笔要伪装成日常细节，不能明显到读者一眼看穿
2. **回收有节奏**：短伏笔5-10章内回收，中伏笔一卷内回收，长伏笔跨卷回收
3. **回收有爽感**：伏笔回收时让读者"原来如此"的恍然，不是平铺直叙的解释

### 本作核心伏笔清单

| 伏笔 | 埋设阶段 | 回收阶段 | 备注 |
|------|---------|---------|------|
| 陆渊混沌灵根变异 | 开局 | 金丹期解封 | 主线 |
| 父亲陆天行血脉印记 | 开局 | 金丹期后逐步 | 家仇线 |
| 大伯谋害父亲真相 | 开局暗示 | 金丹期揭露 | 家仇线 |
| 天丹宗外来祖先与主角灵魂同源 | 中期暗示 | 后期 | 世界观线 |
| 天渊封印真相 | 中期 | 后期 | 世界观线 |
| 四件至宝合一 | 中期 | 终章 | 主线 |
| 爷爷的真实身份 | 开局暗示 | 终章 | 长线 |
| 父母失踪之谜 | 开局 | 终章后 | 续作伏笔 |
| 混沌塔第六层秘藏 | 中期 | 合体期 | 主线 |
| 红颜知己的身份 | 中期邂逅 | 后期 | 感情线 |

## 九、避坑指南

### 绝对禁止

1. **不写纯虐主**：主角可以输，但必须很快找回场子，不让读者憋屈过夜
2. **不写圣母**：对敌人仁慈就是对读者残忍
3. **不写注水**：每章必须有推进，宁可短不可水
4. **不写重复**：不重复升级流程、不重复打脸套路、不重复装逼模式
5. **不写崩人设**：主角性格前后一致，配角不工具人化
6. **不写降智反派**：反派不为了主角装逼而强行变蠢
7. **不写后宫**：感情线单一，不搞暧昧流水账
8. **不写大段设定科普**：设定融入剧情，不写百科全书式段落

### 容易翻车的点

1. **程序员梗别过头**：现代梗是调剂不是主菜，每章最多1-2处，别变成段子集
2. **塔灵别太抢戏**：塔灵是辅助不是主角，戏份控制在每章1-2句互动
3. **大模型别太强**：大模型是工具不是上帝，要有明确限制和失误
4. **回家执念别太丧**：想家是动力不是负能量，每次想念后立刻转化为行动
5. **幽默别尬**：幽默要符合情境，强行抖机灵不如不抖
6. **装逼别太早**：前期主角实力不足，装逼必翻车，藏拙为主
7. **女角别工具人**：红颜知己要有独立人格和剧情，不只是主角的附属

## 十、章节自检清单

每章完稿后，对照以下清单检查：

- [ ] 本章是否推进了至少一项（修为/情报/关系/爽点）？
- [ ] 本章是否有至少一个小爽点或信息揭露？
- [ ] 本章结尾是否留了钩子？
- [ ] 主角的幽默是否自然不尬？
- [ ] 程序员梗/塔灵互动是否控制在合理篇幅？
- [ ] 是否有注水或重复内容？
- [ ] 对话是否每句都有信息量或性格？
- [ ] 战斗/修炼描写是否嵌在剧情中而非独立堆砌？
- [ ] 反派行为是否符合其逻辑？
- [ ] 本章是否与主线（回家/家仇/天渊之谜）有关联或推进？

> **原则**：若某章无法通过上述检查，宁可重写不可将就。节奏一旦垮了，后续很难拉回来。
