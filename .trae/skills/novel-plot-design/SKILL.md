# 修仙 / 玄幻网文 · 剧情设计与行文方法学

> **来源**：用户提供《小说剧情设计行文参考.md》整合为 skill。
> **性质**：纯方法学，不绑定任何具体作品，可套用于任意世界观与设定。
> **适用**：升级爽文、宗门流、丹道流、系统流、废柴逆袭、长生流等同类型作品。
> **与 novel-chapter-writer 的关系**：本 skill 是通用**设计层**（剧情骨架 / 体系 / 赛制 / 人物 / 去 AI 味），novel-chapter-writer 是**执行层**（特定作品的章节撰写 + 硬约束）。设计新卷 / 新赛事 / 新人物时先查本 skill，落笔写章节时再查 novel-chapter-writer。作品专属硬约束（字数 / 破折号 / 违禁词 / 金手指不退化等）一律以 novel-chapter-writer 为准。

## 何时调用本 skill

- 设计新卷的剧情骨架、冲突递进路线
- 设计世界观体系（境界 / 异象 / 排行物）
- 设计比武 / 赛事 / 秘境赛制
- 塑造新配角、给老配角加记忆点
- 行文卡壳、需要节奏 / 去 AI 味参考
- 章后自检（用 §8 清单逐条对照）

## 0. 这个流派共同的骨架

无论具体设定如何，主流修仙网文都共享一套可被复用的骨架：

1. **有一个可量化的实力刻度**（品阶、星级、数值、层数），让读者随时知道"主角现在多强"。
2. **每一次变强都绑定稀缺资源**（丹药、秘境名额、异宝、传承），形成"打怪—得宝—突破"闭环。
3. **冲突层层放大**：个人 → 家族/宗门 → 阵营 → 天下，对手层级随卷升级。
4. **比武 / 赛事 / 秘境是剧情引擎**，不是点缀，而是推动成长的齿轮。

> 记住一句话：**爽文的本质是"可感知的成长 + 有代价的胜利"**。体系让成长可见，资源让胜利有味，冲突让过程有张力。

## 1. 世界观与体系设计：把"实力"变成可感知的东西

**核心原则**：任何抽象境界都必须配一个**可视化标志**。读者记不住"某某境第九重"，但能记住"星辰破万会化星斗""丹成生雷"。

| 设计动作 | 做法 | 通用示例 |
|----------|------|----------|
| 给境界配刻度 | 用一品~九品 / 层数 / 星级做量化 | 炼丹师分九品，再上设"宗师/圣/帝"称号级 |
| 给境界配异象 | 突破或发力的外在表现 | 高品质丹药生"丹纹→丹云→丹雷"三级异象 |
| 给天赋配可视物 | 用颜色 / 数量 / 光晕区分强弱 | 试炼莲台以"九品九彩"论，彩数越高机缘越多 |
| 立"排行物" | 让实力有公开可比的榜单 | 火榜 / 战力碑 / 天骄榜，可被挑战、被超越 |
| 让数值有后果 | 量化指标直接决定战力或上限 | 根基越厚，未来开辟的"府/海"数量越多 |

**设计方法清单**：

- 每设一个大境界，立刻补一句"它的外在表现是什么"。
- 用**品阶 / 颜色 / 数量 / 星光 / 雷声**做刻度，前后必须一致、可比较。
- 体系中至少立一个**公开排行物**，便于制造"登榜/被超越/复仇排名"的爽点。
- 量化指标要**能决定结局**（影响战力上限、突破成功率），而非只是装饰。

> **模板**：境界 = 名称 + 量化指标 + 可视异象 + 排行/碑。

## 2. 剧情结构：资源锚定 + 冲突递进

### 2.1 比武链模式（选拔 → 封顶 → 掠夺）

把比武设计成一条**层层选拔、stakes 递进**的链：

- **入门选拔**：入门级精英赛，魁首得稀缺资源 → 制造早期目标。
- **封顶之战**：各分支/各峰各自决出首席/第一 → 代表荣誉与最高战力。
- **跨域掠夺战**：顶层强者互挑战夺对方核心资源（功法、传承、领地）→ 以战养战。

层层放大 stakes，把个人成长、本势力荣誉、跨势力资源全绑进比武。

### 2.2 同心圆 / 升级曲线模式（舞台与对手逐卷放大）

冲突呈"由内向外"扩张：

终极宿怨（如灭门/夺宝之仇）→ 家族/师门内斗 → 行业争锋（赛事）→ 势力博弈（幕后集团）→ 各方强敌（单元剧 BOSS）。

舞台与对手层级**逐卷放大**：

| 阶段 | 舞台 | 对手层级 |
|------|------|----------|
| 初期 | 家族 / 小城 | 族内天才、地方同行 |
| 发展 | 一州 / 一域 | 赛事群英、同代天骄 |
| 进阶 | 大域 / 古地 | 魔道、地方第一人 |
| 高潮 | 大陆中心 | 顶尖势力、圣地传人 |
| 巅峰 | 诸天 / 万界 | 幕后黑手、各方至尊 |

### 2.3 资源锚定（通用）

每一次大比 / 突破都绑定**实利**，让"赢"有重量：选拔魁首赏核心丹药；团队战成绩决定秘境名额；秘境里抢决定未来上限的异宝。资源越稀缺、越直接关联"破境/续命/窥圣"，驱动力越强。

### 2.4 秘境 = 副本

秘境不是观光，是**多阵营抢点**：把"修炼瓶颈"转化为"可争夺的地图资源"（玉璧、火莲、矿脉），个体战服务于阵营战。

> **模板**：每一卷 = 一个更大舞台 + 更难对手 + 更稀缺资源 + 一场赛事/秘境。

## 3. 比试 / 赛事设计：可复用赛制

| 赛制 | 看点 | 适用场景 | 可改编点 |
|------|------|----------|----------|
| **抽签同题赛** | 公平竞技 + 制造宿命对手 | 公平对决、考基础 | 换考题、加限时、加禁术 |
| **多层淘汰赛** | 初赛辨材→复赛斗技→决赛高阶 | 长篇赛事弧 | 做赛事骨架，只换名称与反派 |
| **资源赌斗** | 赌约 + 契约增高压 | 势力博弈 | 赌领地/资源/性命，立"输了必付"铁律 |
| **形态化斗** | 把力量化为兽形/器形对撞 | 视觉爽点 | 换异火/本命物，加吞噬进化 |
| **禁力神魂考** | 禁用主战力、考隐性实力 | 考主角隐藏底牌 | 禁某力量 + 监视物防作弊 |
| **团战抢点** | 多阵营混战争资源点 | 秘境/大战 | 分玉璧/莲台/矿脉，阵营结盟与背叛 |
| **跨域掠夺战** | 胜者夺对方核心资源 | 顶层冲突 | 夺功法/传承/领地，以战养战 |

**设计铁律**：每场比武必须同时具备——**stakes（实利）+ 看点（异象/反转）+ 晋级（通向更大舞台）**。缺一样，这场戏就只是过场。

## 4. 人物塑造：功能化 + 反差

### 4.1 主角：动机要硬，底牌要藏

- **动机具体到一个物件 / 一个人**：不是空泛"我要变强"，而是"母寿将尽，必夺续命之物""师门被灭，必手刃仇人"。具体动机才能撑起长线。
- **藏底牌**：明面从弱势起步，暗握前世记忆 / 特殊体质 / 隐藏传承。不到关键时刻不亮，亮则惊全场。

### 4.2 配角要有"功能"，别脸谱

| 功能 | 常见形态 | 作用 |
|------|----------|------|
| 导师 / 长辈 | 护短毒舌的老怪、神秘深不可测的师父 | 给资源、给信息、兜底 |
| 羁绊 / 红颜 | 身世成谜的同伴 | 情感线、秘密线 |
| 常态宿敌 | 同代第一人、敌对势力少主 | 长期对标、阶段性决战 |
| 单元剧 BOSS | 每卷一个强敌 | 稳定爆点，多呼应隐藏设定 |
| 质疑者 / 垫脚石 | 同门天才、看不起主角的长辈 | 制造阻力、提供打脸爽点 |

### 4.3 反差人设（去扁平的关键）

- **最强者最佛系**：顶层强者却懒散怕麻烦，反差制造记忆点。
- **主角扮猪**：被所有人轻视，实则根基深厚，发力时刷新认知。
- 每个重要配角给一个**记忆点 + 小动作 / 口头禅**（挠头、幸灾乐祸、护短），比堆设定更立人。

> **原则**：配角不是"工具人"，而是"有功能的活人"——他出场要么推动、要么阻碍、要么给资源、要么制造笑点。

## 5. 行文风格：节奏与质感

- **短句驱动、动作优先**：少用"他感到很紧张"，多写"他手指在剑柄上摩挲了三遍"。
- **对话推进剧情**，不靠旁白解释设定；让信息在人物交锋中自然带出。
- **章末抛钩子**：留一个未解之谜或反转，不写"这一章讲了……"式的总结。
- **具体感官代替空泛形容词**：写"如婴儿般的玉果""晶莹剔透、颗颗饱满的米"，不写"极其珍贵的宝物"。
- **口语化 / 反差萌**：强者犯懒、对手幸灾乐祸，用生活化短句。
- **信息密度控制**：紧张处碎句碎段；设定说明一句带过，**边写边揭，不一次性倒给读者**。

> ⚠️ **与 novel-chapter-writer 的衔接**：本节"章末抛钩子"是通用建议。若作品有专属约束（如《修仙：我的灵海有AI大模型》要求"结尾不是一定要钩子，可自然收束"），**以作品专属约束为准**——通用方法学让位于作品硬约束。

## 6. 去 AI 味清单（正反对照）

**AI 味典型症状**：

- 套话开头："在修仙界，实力为尊……""这是一个……的世界"。
- 过度解释 + 总结式金句堆砌（每段都"起承转合"圆满）。
- 对仗工整的排比、书面报告体。
- 视角飘移、说明代替展示。
- 信息密度均一，全程一个节奏。
- 人物说话一个腔调（都像旁白在汇报）。
- 滥用"仿佛 / 似乎 / 无疑 / 可谓是 / 不得不说"。
- 每章结构雷同、缺钩子。

**去味技法（写时逐条对照）**：

1. **用动作 / 细节代替形容词**：写"他手指在剑柄上摩挲了三遍"而非"他十分紧张"。
2. **让人物说人话**：带情绪、带省略、带打断；不用报告体。
3. **留白与悬念**：不一次性交底，章末抛钩子。
4. **制造反差与意外**：强者犯懒、弱者藏刀、胜利得来别太顺。
5. **控制节奏**：紧张处碎句碎段；平叙也不冗长，设定一句带过。
6. **删空词**：少用"可谓 / 无疑 / 显然 / 仿佛 / 似乎"，多用具体名词。
7. **配角记忆点**：给口头禅 / 小动作，避免全员一个声线。
8. **数字前后一致**：品阶、数量、实力对比要自洽可量化。
9. **允许中断与余韵**：不必每段圆满，留呼吸感。
10. **少用成语排比堆砌**，宁可朴白有力。

> ⚠️ **行文大忌·列计划再逐项实现**：小说不是写代码，绝不能"先罗列接下来的事，再逐项执行"。谋定而后动是好的，但计划要随剧情自然带出，不能写成清单。病句示例：「优先级：确认潮信 / 备丹药 / 拔人 / 磨底牌」「他排了条线：备丹 / 淬体 / 壮魂 / 磨招」——这是说明书腔，要改成随剧情推进自然浮现。修炼段尤其要跳出来用上帝视角看大致路线和收获，少钻细节。

## 7. 一章示范（短例，展示上述风格）

> 残雪压着断崖，那少年盘坐的青石上凝了一层薄霜。
> 他没动。气府里那颗新凝的星辰，正一点点把寒雾往外推。
> "星辰化斗……"对面的声音传来，带着点不敢信，"你藏着呢。"
> 少年睁眼，没答。他指尖在膝头敲了一下——这是给暗处同伴的信号。
> 崖下雾里传来一声低吼。
> 对面人脸色变了："你早算计好了？"
> "算计谈不上。"少年起身，金光自天灵冲起，隐隐衍成一片星斗，"只是你刚才那句'就你这点根基'，我记下了。"
> 风停了一瞬。
> 然后，整座断崖的雪，齐齐往下落了一寸。

*示范点*：动作开场（残雪 / 青石 / 摩挲）、对话推进、反差（藏底牌）、具体名词（星斗 / 寒雾）、章末钩子（雪落一寸）、零解释、零空词。

## 8. 写作后自检清单

- [ ] 这一章有没有**实利 stakes**（资源 / 名额 / 仇 / 命）？
- [ ] 有没有**新信息 / 反转 / 钩子**？（或按作品约束：自然收束亦可）
- [ ] 人物说话是否**各有腔调**（带情绪、带打断）？
- [ ] 有没有"说明代替展示"的段落？
- [ ] 有没有 AI 高频空词（可谓 / 无疑 / 仿佛 / 显然）？
- [ ] 有没有"列计划再逐项实现"的清单式行文？
- [ ] 实力 / 品阶是否**前后一致、可量化**？
- [ ] 配角是否有**记忆点 / 功能**，而非工具人？
- [ ] 章末是否留了**钩子或自然余韵**，而非圆满总结？
- [ ] 节奏是否有松紧（紧张处碎句）？

## 9. 与 novel-chapter-writer 的协作

| 场景 | 用哪个 skill |
|------|--------------|
| 设计新卷剧情骨架、冲突递进路线 | **本 skill**（§2 剧情结构） |
| 设计世界观体系（境界/异象/排行物） | **本 skill**（§1 体系设计） |
| 设计比武/赛事/秘境赛制 | **本 skill**（§3 赛制表） |
| 塑造新配角、加记忆点 | **本 skill**（§4 人物塑造） |
| 行文卡壳、需要节奏/去AI味参考 | **本 skill**（§5 行文风格 + §6 去AI味） |
| 章后自检（通用项） | **本 skill**（§8 自检清单） |
| **落笔写特定作品的某一章** | **novel-chapter-writer**（含作品专属硬约束） |
| **校验字数/破折号/违禁词/金手指参数** | **novel-chapter-writer**（作品专属） |
| **查特定作品的人物当前状态/剧情走向** | **novel-chapter-writer**（作品专属） |

**冲突仲裁原则**：

- **通用方法学**（本 skill）与**作品专属约束**（novel-chapter-writer）冲突时，**作品约束优先**。
- 例：本 skill §5 建议"章末抛钩子"，但《修仙：我的灵海有AI大模型》要求"结尾不是一定要钩子，可自然收束"——按作品约束走。
- 例：本 skill §6 列出去 AI 味通用清单，但作品的违禁词表（Bug/Debug/算法/调参等程序员黑话）以 novel-chapter-writer 为准。
- 例：本 skill §0 说"可量化实力刻度"，但具体数值（180 节点 / 神识三百丈 / 筑基中期）以 novel-chapter-writer 的金手指当前状态为准。
